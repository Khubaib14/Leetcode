# Reverse each word in a given string
## Medium 
<div class="problem-statement" style="user-select: auto;">
                <p style="user-select: auto;"></p><p style="user-select: auto;"><span style="font-size: 18px; user-select: auto;">Given a String. Reverse each word in it where the words are separated by dots.</span></p>

<p style="user-select: auto;"><span style="font-size: 18px; user-select: auto;"><strong style="user-select: auto;">Example 1:</strong></span></p>

<pre style="position: relative; user-select: auto;"><span style="font-size: 18px; user-select: auto;"><strong style="user-select: auto;">Input:
</strong>S = "i.like.this.program.very.much"
<strong style="user-select: auto;">Output:</strong> 
i.ekil.siht.margorp.yrev.hcum
<strong style="user-select: auto;">Explanation</strong>: 
The words are reversed as
follows:"i" -&gt; "i","like"-&gt;"ekil",
"this"-&gt;"siht","program" -&gt; "margorp",
"very" -&gt; "yrev","much" -&gt; "hcum".</span><div class="open_grepper_editor" title="Edit &amp; Save To Grepper" style="user-select: auto;"></div></pre>

<p style="user-select: auto;"><span style="font-size: 18px; user-select: auto;"><strong style="user-select: auto;">Example 2:</strong></span></p>

<pre style="position: relative; user-select: auto;"><span style="font-size: 18px; user-select: auto;"><strong style="user-select: auto;">Input</strong>: 
S = "pqr.mno"
<strong style="user-select: auto;">Output:</strong> 
rqp.onm
<strong style="user-select: auto;">Explanation</strong>: 
The words are reversed as
follows:"pqr" -&gt; "rqp" ,
"mno" -&gt; "onm"</span><div class="open_grepper_editor" title="Edit &amp; Save To Grepper" style="user-select: auto;"></div></pre>

<span style="font-size: 18px; user-select: auto;"><strong style="user-select: auto;">Your Task:</strong><br style="user-select: auto;">
